# ConsoleCapture

## 何のソフトなの？

このプログラムは、 コンソール出力(System.out, System.err)を文字列に
キャプチャするためのライブラリです。

Junit等でコンソール出力をログで目視する代わりに作成してみました。

src/test/java/com/hayatsukikazumi/coc/Smpletest.java に大まかな使い方
のイメージを示しました。

## 動作条件

Java（バージョン1.7以降の実行環境）をインストールしたマシン
（Pure JavaなのでOSは問いません）
